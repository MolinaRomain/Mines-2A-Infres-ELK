# TP ELK Stack

Ce dossier contient le sujet du TP ELK Stack et le corrigé/barème enseignant.

Fichiers :

- `TP_ETUDIANTS.md` : énoncé à distribuer aux étudiants.
- `CORRECTION_BAREME.md` : corrigé, critères de validation et barème.

Le TP est conçu pour être réalisé dans un cluster local `kind`, avec des manifests Kubernetes simples et une stack Elastic basée sur Elasticsearch, Logstash, Kibana et Filebeat.
